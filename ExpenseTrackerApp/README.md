A modern Android application developed using Java in Android Studio that helps users manage 
their personal finances by tracking daily income and expenses. The app stores transaction data
locally using SQLite, provides a transaction history, calculates account summaries, and displays
spending analysis using interactive charts.

✨ Features
➕ Add income and expense transactions
📋 View complete transaction history
🗑️ Delete transactions
💰 Automatically calculate total income, total expenses, and current balance
📊 Monthly summary with Pie Chart and Bar Chart
📅 Automatically records the transaction date
📂 Categorize transactions (Food, Shopping, Salary, Bills, etc.)
📱 Clean and user-friendly Material Design interface
🛠️ Technologies Used
  Java
  Android Studio
  SQLite Database
  RecyclerView
  CardView
  Material Design Components
  MPAndroidChart
